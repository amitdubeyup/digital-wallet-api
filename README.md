## Quick Start

  Clone the repository:

```bash
$ git clone https://github.com/amitdubeyup/wallet
```

  Goto project directory:

```bash
$ cd wallet
```

  Install dependencies:

```bash
$ npm install
```

  Start the app:

```bash
$ npm start
```

  View the app at: http://localhost:3000
